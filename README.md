# GraphQL API Gateway

A GraphQL API gateway built with Apollo Server.

## Features

- GraphQL queries and mutations
- Type system
- Resolvers
- Apollo Server
- Playground included

## Tech Stack

- **Backend**: Node.js
- **GraphQL**: Apollo Server
- **Schema**: GraphQL SDL

## Project Structure

\`\`\`
graphql-api-gateway/
├── src/
│   ├── schema.js        # GraphQL schema
│   └── server.js        # Apollo Server
└── package.json
\`\`\`

## Installation

\`\`\`bash
npm install
\`\`\`

## Usage

\`\`\`bash
npm start
\`\`\`

## GraphQL Playground

Visit \`http://localhost:4000\` for GraphQL Playground.
